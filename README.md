# Suica Auto Reader v1.3.0

A small Android demo app for reading Suica/FeliCa balance and recent public history.

## v1.3.0 changes

- Added multilingual UI support: Simplified Chinese, Traditional Chinese, English, Japanese, and Korean.
- Added language selection in Settings: System / 简体中文 / 繁體中文 / English / 日本語 / 한국어.
- The main screen, drawer menu, status messages, settings page, and readable history categories are localized.
- The app still uses the system NFC app chooser and foreground NFC-F reader mode.

## Build

Upload this folder to GitHub and run the included GitHub Actions workflow.
